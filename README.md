# menstRUA — Site Institucional

Site institucional do projeto social **menstRUA**, que atua na luta pela dignidade menstrual no Amazonas por meio da distribuição de kits menstruais, educação menstrual e ativismo.

---

## Tecnologias

- [React 17](https://reactjs.org/)
- [Styled Components](https://styled-components.com/)
- [React Router DOM v6](https://reactrouter.com/)
- [React Icons](https://react-icons.github.io/react-icons/)
- [Create React App](https://create-react-app.dev/)
- [Netlify](https://www.netlify.com/) — deploy e CI/CD

---

## Estrutura de Seções

| Componente | Descrição |
|---|---|
| `Banner` | Hero com SVG customizado, glow radial e caixas de dados estatísticos |
| `Projeto` | Apresentação do projeto com layout de fotos sobrepostas |
| `Impacto` | Números e impacto social do menstRUA |
| `Ações` | Galeria das frentes de atuação: distribuição, educação e ativismo |
| `Integrantes` | Cards das coordenadoras — carrossel no mobile, grid no desktop |
| `Voluntariado` | Galeria de fotos e formulário de inscrição de voluntários |
| `Contribua` | Seção de doação |
| `Mídia` | Aparições na mídia |
| `Menu` | Navegação fixa com scroll suave para as seções |
| `Footer` | Rodapé com links e redes sociais |

## Como rodar localmente

**Pré-requisitos:** Node.js 16+

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm start
```

---

## Build para produção

```bash
npm run build
```

Os arquivos otimizados serão gerados na pasta `/build`.

---

## Deploy

O deploy é feito automaticamente via **Netlify** a cada push na branch `main`. O domínio é gerenciado pela **[Hostinger](https://www.hostinger.com.br/)**.

O site pode ser acessado em: [www.projetomenstrua.com.br](https://www.projetomenstrua.com.br)

## Sobre o menstRUA

O menstRUA é um projeto social que atua em Manaus/AM na luta pela dignidade menstrual, com três frentes principais:

- **Distribuição de kits menstruais** — para pessoas em situação de rua, comunidades periféricas, quilombolas, indígenas e escolas públicas
- **Educação menstrual** — oficinas e rodas de conversa
- **Ativismo menstrual** — projeções em prédios e aplicação de lambe-lambe
